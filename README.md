# Smart Data Management System

A full-stack web application to manage user data using CRUD operations.

## 🚀 Features
- Add users
- View users
- Delete users
- Duplicate email validation

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## 📂 Project Structure
backend/
frontend/

## ▶️ How to Run
1. Install dependencies:
   pip install flask flask-cors

2. Run backend:
   cd backend
   python app.py

3. Open frontend:
   frontend/index.html

## 📌 Output
- Add user
- Display users
- Delete users
