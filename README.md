# Smart Data Management System

This is a CRUD-based web application using Flask and SQLite.

## Features
- Add users
- View users
- Delete users
- Duplicate detection

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## How to Run
1. Install:
   pip install flask flask-cors

2. Run backend:
   python app.py

3. Open frontend:
   index.html
